o-Do Oppgaveliste Webapp
Beskrivelse:
Jeg skal lage en enkel nettside hvor brukeren kan skrive inn oppgaver de må huske. Brukeren kan legge til nye oppgaver, se oppgavene og slette oppgaver som er ferdige.

Applikasjonen kjører på en Raspberry Pi med Flask som webserver og MariaDB for lagring av data.

Hvorfor prosjektet
Jeg trener på programmering (Python)
Jeg setter opp server og database (drift)
Jeg lager veiledning for brukeren (brukerstøtte)
Prosjektet oppfyller kravene i IT-faget
Teknologi
Del	Forklaring
Raspberry Pi	Kjører webappen (vis jeg klarer meg)
Flask (Python)	Webserver 
MariaDB	Lagrer oppgavene
HTML	Viser nettsiden til brukeren
Funksjoner
Legge til oppgave
Se liste over oppgaver
Slette oppgave
Mål for prosjektet
Å lage en enkel, forståelig og brukbar webapp som kan kjøres på nettverket.
